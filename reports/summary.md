# The Emperor's Interactome — summary

AI-predicted protein interactomes now label thousands of pairs "high-confidence," but those error rates are estimated on class-balanced benchmarks and inflate sharply in the real regime of a genome-scale screen, where true interactions are rare. The CM4AI multimodal cell map (Schaffer et al., *Nature* 2025, Ideker lab) — the NIH Bridge2AI *Cell Maps for AI* project (contact PI Trey Ideker, UC San Diego; co-PI Nevan Krogan, Gladstone Institutes / UCSF) — is among the few such maps that can be audited externally, because it ships a raw confidence axis (AlphaFold-Multimer ipTM) with a native random-pair null — the exchangeable calibration set that distribution-free error control requires.

We apply **conformal false-discovery-rate control** to the map's published high-confidence tier, refereed by **DepMap co-essentiality** under a strict purity firewall (the referee never enters any score, label, or calibration set). At q=0.10, **35 of 161 (22%) high-confidence edges fail honest FDR control**. A semi-synthetic experiment on known truth shows the mechanism: a benchmark-tuned cutoff's realized FDR climbs toward 0.9 as interactions grow rare, while conformal control stays bounded near q. The independent referee agrees with the audit's direction — **certified edges are 41% co-essential versus 17% for dropped edges** (permutation p=0.016).

We then interrogate the guarantee itself: we measure the calibration-to-candidate exchangeability violation, show where the marginal guarantee breaks, and report the strongly-certified core as robust to bounded violation far larger than the one present. The certified set is independently enriched for wet-lab physical interactions (IntAct) and recovers held-out complex members (leave-one-member-out, odds ratio 3.3, p<10⁻⁴); a known chromatin-complex member (KANSL3) is recovered as a labeled positive control. The method transfers to a second, genome-scale AI map. Fully reproducible via `make reproduce`.
